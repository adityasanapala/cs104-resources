## Trying Out!

Once you are done with the installation of CLab, try out the following basic Linux commands to familiarize yourself with the terminal.

### Commands to Try
1. pwd Print the current working directory
• Type the command:
pwd
• This command shows the complete path of your current directory.
• Example:
/home/user/myfolder
2. Ls List files and directories
• Type the command:
ls
• This command displays the list of files and directories in your current location.
3. Try combining the commands!
Example worktion
1. Check answers.txt is present using ls.
2. Verify your current directory using pwd.

Your task in this lab is to type 'hello world' in the file answers.txt and submit it.
